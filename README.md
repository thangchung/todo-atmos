# Get starting

- Install `subo` CLI at https://github.com/suborbital/subo
- Build `todo` app

```bash
> cd todos
> subo build .
```

- Run `docker compose`

```bash
> docker compose up
```

- Play with it

Install humao.rest-client vscode extensions, then play with [client.http](client.http)
